## ⚡ 핵심 개념 3가지만 기억하세요

1. **프론트엔드** = 사용자가 보는 화면 (버튼, 입력창, 디자인)
2. **백엔드** = 데이터를 처리하는 서버 (로그인, 저장, 삭제)
3. **데이터베이스** = 실제로 데이터가 저장되는 곳
