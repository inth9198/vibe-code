# 화면 수정하기 - 프론트엔드 실전

## 🎯 목표
버튼, 텍스트, 색상 등 보이는 부분을 내 맘대로 바꾸기

## 🎨 어디를 수정하면 될까?

### 1. 텍스트 바꾸기

**상단 로고 "Vibe" 바꾸기**

파일: `frontend/src/components/Header.jsx`

```jsx
// 찾기: 이 부분을
<h1 className="logo">Vibe</h1>

// 바꾸기: 원하는 이름으로
<h1 className="logo">내 사이트</h1>
```

**로그인 페이지 제목 바꾸기**

파일: `frontend/src/pages/Login.jsx`

```jsx
// 찾기
<h2>로그인</h2>

// 바꾸기
<h2>어서오세요! 로그인하세요</h2>
```

### 2. 색상 바꾸기

**전체 테마 색상 변경**

파일: `frontend/src/App.css`

```css
/* 주요 색상 찾아서 바꾸기 */
:root {
  --primary-color: #667eea;    /* 메인 색상 */
  --secondary-color: #764ba2;  /* 보조 색상 */
  --text-color: #333;          /* 글자 색상 */
  --bg-color: #f7fafc;         /* 배경 색상 */
}

/* 예: 파란색 테마로 바꾸기 */
:root {
  --primary-color: #3b82f6;    /* 파란색 */
  --secondary-color: #1e40af;  /* 진한 파란색 */
}
```

**버튼 색상 바꾸기**

파일: `frontend/src/App.css`

```css
/* 버튼 스타일 찾기 */
.btn-primary {
  background: var(--primary-color);
  color: white;
}

/* 직접 색상 지정하기 */
.btn-primary {
  background: #ff6b6b;  /* 빨간색 버튼 */
  color: white;
}
```

### 3. 버튼 추가하기

**게시물 목록에 새로고침 버튼 추가**

파일: `frontend/src/pages/PostList.jsx`

```jsx
// 함수 추가 (컴포넌트 안에)
const handleRefresh = () => {
  fetchPosts();  // 게시물 다시 불러오기
};

// JSX에 버튼 추가 (return 안에)
<div className="actions">
  <Link to="/new">
    <button className="btn-primary">새 글 쓰기</button>
  </Link>
  {/* 여기에 새로운 버튼 추가 */}
  <button onClick={handleRefresh} className="btn-secondary">
    새로고침
  </button>
</div>
```

### 4. 입력창 추가하기

**게시물에 "카테고리" 필드 추가하기**

파일: `frontend/src/pages/PostForm.jsx`

```jsx
// 1. 상태 추가 (다른 useState 옆에)
const [category, setCategory] = useState('');

// 2. 제출할 때 포함시키기 (handleSubmit 함수 안에)
const postData = {
  title,

  category  // 추가!
};

// 3. JSX에 입력창 추가 (다른 input 옆에)
<div className="form-group">
```

## 🎯 React 기본 패턴 - 복붙해서 쓰세요!

### 패턴 1: 버튼 클릭 처리

```jsx
// 클릭하면 실행될 함수 만들기
const handleClick = () => {
  alert('버튼이 클릭되었습니다!');
};

// 버튼에 연결하기
<button onClick={handleClick}>클릭하세요</button>
```

### 패턴 2: 입력창 값 저장하기

```jsx
// 1. 상태 만들기
const [inputValue, setInputValue] = useState('');

// 2. 입력창 만들기
<input
  type="text"
  value={inputValue}
  onChange={(e) => setInputValue(e.target.value)}
/>

// 3. 입력한 값 사용하기
<p>입력한 값: {inputValue}</p>
```

### 패턴 3: 목록 보여주기

```jsx
// 배열 데이터
const items = ['사과', '바나나', '딸기'];

// 화면에 표시
<ul>
  {items.map((item, index) => (
    <li key={index}>{item}</li>
  ))}
</ul>
```

### 패턴 4: 조건부 보여주기

```jsx
// 로그인 상태에 따라 다르게 표시
{user ? (
  <p>안녕하세요, {user.username}님!</p>
) : (
  <p>로그인이 필요합니다.</p>
)}
```

## 💡 실전 예제: "좋아요" 버튼 추가하기

### 1단계: 게시물에 좋아요 수 표시

파일: `frontend/src/pages/PostList.jsx`

```jsx
// 각 게시물 표시 부분에 추가
<div className="post-card">
  <h3>{post.title}</h3>
  <p>{post.content}</p>
  
  {/* 좋아요 수 표시 추가 */}
  <div className="post-footer">
    <span>❤️ {post.likes || 0} 좋아요</span>
  </div>
</div>
```

### 2단계: 좋아요 버튼 추가

```jsx
// 좋아요 클릭 함수
const handleLike = async (postId) => {
  try {
    await postsAPI.likePost(postId);  // API 호출
    fetchPosts();  // 목록 새로고침
  } catch (error) {
    console.error('좋아요 실패:', error);
  }
};

// 버튼 추가
<div className="post-footer">
  <button 
    onClick={() => handleLike(post._id)}
    className="btn-like"
  >
    ❤️ 좋아요
  </button>
  <span>{post.likes || 0}</span>
</div>
```

## ⚠️ 주의사항

### 1. 저장하면 자동으로 반영됩니다
파일 저장하면 브라우저가 자동으로 새로고침됩니다.

### 2. 오류가 나면 터미널을 확인하세요
빨간색 에러 메시지가 뜨면 뭐가 잘못됐는지 알려줍니다.

### 3. 중괄호 { } 개수를 맞추세요
하나라도 빠지면 전체가 안 됩니다!

### 4. 백엔드도 수정해야 할 수 있습니다
새로운 데이터 필드를 추가하면 백엔드도 같이 수정해야 합니다.
